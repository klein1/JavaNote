## 排序效率

| 算法 | 平均速度           | 最坏             |
| ---- | ------------------ | ---------------- |
| 冒泡 | O(n<sup>2</sup>)   | O(n<sup>2</sup>) |
| 快速 | O(nlogn)           | O(n<sup>2</sup>) |
| 选择 | O(n<sup>2</sup>)   | O(n<sup>2</sup>) |
| 堆排 | O(nlogn)           | O(nlogn)         |
| 插入 | O(n<sup>2</sup>)   | O(n<sup>2</sup>) |
| 希尔 | O(n<sup>3/2</sup>) | O(n<sup>2</sup>) |
| 合并 | O(nlogn)           | O(nlogn)         |
|      |                    |                  |
